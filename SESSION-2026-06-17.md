## 4. Agreed architecture

Data model adds a tenant layer above existing tables:
```
organizations (tenant) ─ users (role) ─ farms (NEW, gains org_id) ─ zones(field_zones, gains org_id/farm_id)
                                                                       └─ health_indices, alerts, satellite_images
```
A FastAPI dependency resolves current org from JWT and filters **every** query by `org_id`.
Pair with a test asserting cross-tenant access returns 404.

Prod topology:
```
Contabo VPS (Debian): Docker Compose [api · processor · postgres · frontend(nginx)]  +  Caddy → auto HTTPS
```
Same compose file on NAS (dev) and Contabo (prod); only env + domain differ.
